# bootstrap-material-design-sass

A Sass implementation of material design on bootstrap.

This is a lightweight implementation which did not provides any script library.
A lot of different framework exists to implements the animation so this project
is solely about providing the SCSS, CSS part.

## Contributing

Contribution to this project are very welcome.

Push request will be accepted quickly, but a simple bug report is already a good
things.
